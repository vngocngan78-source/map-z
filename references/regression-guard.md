# 回归测试规范

## 概述

每次更新 SKILL.md 或核心模板后，需通过以下测试用例验证生成效果没有退化。

## 基础测试用例

### TC-01：全国默认地图
```
输入: "帮我生成一个全国GDP地图"
期望:
  ✅ 31个省份全部渲染
  ✅ 流光边界动画正常
  ✅ 侧边栏显示排名
  ✅ 鼠标悬停 Tooltip 正常
  ✅ 点击省份可以下钻
```

### TC-02：飞线地图
```
输入: "生成从上海出发到各主要城市的飞线地图"
期望:
  ✅ 飞线弧度正确（不是直线）
  ✅ 飞线流动动画正常
  ✅ 飞线粗细根据 value 变化
  ✅ 下钻时飞线自动隐藏
  ✅ 返回全国时飞线恢复
```

### TC-03：省级下钻
```
输入: 点击广东省
期望:
  ✅ 加载广东省市级地图
  ✅ 面包屑显示"全国 › 广东"
  ✅ "返回上级"按钮出现
  ✅ 视角切换流畅
  ✅ 再次点击市可继续下钻
```

### TC-04：配色切换
```
输入: theme: 'orange'
期望:
  ✅ 省份颜色变为橙色系
  ✅ 边界流光变为橙色
  ✅ Tooltip 边框颜色变化
  ✅ 侧边栏数据条颜色变化
```

### TC-05：空数据容错
```
输入: data: []
期望:
  ✅ 地图正常渲染（省份显示暗色）
  ✅ 不报错，不崩溃
  ✅ 侧边栏显示"暂无数据"
```

## 视觉验收标准

| 项目 | 标准 |
|------|------|
| 流光亮度 | 有明显呼吸感，不闪烁 |
| 3D高度差 | 高值省份明显高于低值省份 |
| 飞线弧度 | 最高点约为起终点距离的 20% |
| 加载时间 | 正常网络下 < 3秒 |
| 移动端 | 可触摸缩放，不错位 |

## 如何运行测试

1. 下载 `examples/demo.html`
2. 在 Chrome 中打开
3. 逐项检查上述测试用例
4. 截图存入 `docs/screenshots/` 文件夹
